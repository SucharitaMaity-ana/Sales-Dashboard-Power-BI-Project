# 📊 Sales Dashboard | Power BI

![Sales Dashboard](Sales.png)

## 📌 Project Overview

This project is an interactive **Sales Dashboard** developed using **Microsoft Power BI** to analyze sales performance across different regions, product categories, and time periods. It helps businesses monitor key performance indicators (KPIs), identify sales trends, and make data-driven decisions through interactive visualizations.

---

## 📸 Dashboard Preview

![Sales Dashboard](Sales.png)

---

## 🎯 Objectives

- Analyze overall sales performance.
- Monitor revenue and order quantity.
- Compare sales by region, city, product category, and year.
- Track monthly and daily sales trends.
- Build an interactive dashboard for business reporting.

---

## 🛠️ Tools & Technologies

- Microsoft Power BI
- Power Query
- DAX (Data Analysis Expressions)
- Data Modeling

---

## 📈 Dashboard Features

- 📌 Total Sales KPI
- 📌 Order Quantity
- 📌 Average Price
- 📌 Unit Cost
- 📌 Sales by Region & City (Map)
- 📌 Sales by Product Category
- 📌 Sales by Product Colour
- 📌 Monthly Sales Trend
- 📌 Daily Sales Trend
- 📌 Year & Month Filters
- 📌 Interactive Slicers

---

## 💡 Key Insights

- Identified top-performing regions and cities.
- Compared product category performance.
- Analyzed monthly and daily sales patterns.
- Built interactive filters for dynamic reporting.
- Enabled quick business decision-making through KPIs.
